Interface Thenable<T>

Thenable 是 ES6 promises、Q、jquery.Deferred、WinJS.Promise 等之间的共同接口。 该 API 不假设使用哪个 promise 库，因此可以重用现有代码而无需迁移到特定的 promise 实现。 但是，我们建议在此编辑器中使用原生 promises。
